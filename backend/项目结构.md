项目结构(后端) - Salvo 简化架构
MyTodoApp-server/
├── src/
│   ├── main.rs                   // 程序入口
│   ├── models/                   // 数据模型
│   │   ├── user.rs               // 用户信息
│   │   ├── task.rs                // 任务信息
│   │   ├── team.rs                // 团队信息
│   │   ├── user_log.rs            // 用户日志
│   │   ├── task_log.rs            // 任务日志
│   │   ├── team_log.rs            // 团队日志
│   │   └── user_settings.rs       // 用户设置
│   ├── db/                       // 数据库操作层
│   │   ├── pool.rs                // 数据库连接池
│   │   ├── migrations.rs           // 数据库迁移
│   │   ├── db_user.rs              // 用户相关数据库操作
│   │   ├── db_task.rs              // 任务相关数据库操作
│   │   ├── db_team.rs              // 团队相关数据库操作
│   │   ├── db_sub_team.rs          // 子团队数据库操作
│   │   ├── db_sub_team_member.rs   // 子团队成员数据库操作
│   │   ├── db_user_log.rs          // 用户日志数据库操作
│   │   ├── db_task_log.rs          // 任务日志数据库操作
│   │   └── db_team_log.rs          // 团队日志数据库操作
│   ├── middleware/                // 中间件
│   │   ├── auth.rs                 // 认证中间件
│   │   ├── logging.rs              // 日志中间件
│   │   └── permission.rs           // 权限中间件
│   └── utils/                     // 工具函数
│       ├── utils_passwd.rs         // 密码工具：哈希、验证
│       ├── id_generator.rs        // ID 生成工具：雪花 ID
│       ├── jwt.rs                  // JWT 工具
│       └── validator.rs            // 数据验证工具
├── Cargo.toml
└── readme.md

请求处理流程（Salvo 简化架构）
    ↓
main.rs / router                   // 路由定义（在 main.rs 中）
    ↓
handlers (内联在 main.rs 中)        // 请求处理
    ↓
db/*                                // 数据库操作 ← 当前已实现
