# Skill: Leptos CSR 模态框（Modal）完整实现

---

## 1. 概述

在 Leptos 框架的 CSR（Client-Side Rendering）模式下，利用 Rust 的信号系统（Signal）和组件宏（`#[component]`）构建一个功能完备、可复用的前端模态框组件。涵盖：状态管理、动画过渡、键盘事件、点击遮罩关闭、插槽内容分发、以及多种变体支持。

### 3.2 核心 Modal 组件（modal.rs）

```rust
use leptos::*;
use wasm_bindgen::prelude::*;
use web_sys::{KeyboardEvent, Event};

/// Modal 尺寸变体
#[derive(Clone, Debug, PartialEq)]
pub enum ModalVariant {
    Default,
    Alert,
    Fullscreen,
}

impl ModalVariant {
    fn css_class(&self) -> &str {
        match self {
            ModalVariant::Default => "",
            ModalVariant::Alert => "modal-alert",
            ModalVariant::Fullscreen => "modal-fullscreen",
        }
    }
}

/// Modal 组件属性
#[component]
pub fn Modal(
    /// 是否显示 Modal
    show: ReadSignal<bool>,
    /// 控制显示/隐藏的写端
    set_show: WriteSignal<bool>,
    /// 标题
    #[prop(optional)]
    title: Option<String>,
    /// 尺寸变体
    #[prop(default = ModalVariant::Default)]
    variant: ModalVariant,
    /// 点击遮罩是否关闭
    #[prop(default = true)]
    close_on_overlay: bool,
    /// 按 Esc 是否关闭
    #[prop(default = true)]
    close_on_escape: bool,
    /// 关闭后的回调
    #[prop(optional)]
    on_close: Option<Callback<()>>,
    /// 确认回调
    #[prop(optional)]
    on_confirm: Option<Callback<()>>,
    /// 确认按钮文本
    #[prop(default = "确认".to_string())]
    confirm_text: String,
    /// 是否显示底部按钮栏
    #[prop(default = true)]
    show_footer: bool,
    /// 是否显示确认按钮
    #[prop(default = true)]
    show_confirm: bool,
    /// 确认按钮样式
    #[prop(default = "btn btn-primary".to_string())]
    confirm_class: String,
    /// 子内容
    children: Children,
) -> impl IntoView {
    // 用于控制动画的内部状态：先挂载 DOM，再触发 active class
    let (mounted, set_mounted) = create_signal(false);
    let (active, set_active) = create_signal(false);

    // 当 show 变为 true 时：先挂载 → 等一帧 → 添加 active class
    // 当 show 变为 false 时：先移除 active class → 等动画结束 → 卸载
    create_effect(move |_| {
        if show.get() {
            // 挂载
            set_mounted.set(true);
            // 锁定 body 滚动
            lock_body_scroll(true);
            // 下一帧添加 active（触发 CSS transition）
            request_animation_frame(move || {
                set_active.set(true);
            });
        } else {
            // 先移除 active，触发关闭动画
            set_active.set(false);
            // 等待动画结束后卸载
            set_timeout(
                move || {
                    set_mounted.set(false);
                    lock_body_scroll(false);
                },
                std::time::Duration::from_millis(300),
            );
        }
    });

    // 关闭函数
    let close_modal = Callback::new(move |_: ()| {
        set_show.set(false);
        if let Some(cb) = on_close {
            cb.call(());
        }
    });

    // Esc 键处理
    let handle_keydown = move |evt: KeyboardEvent| {
        if close_on_escape && evt.key() == "Escape" {
            evt.prevent_default();
            close_modal.call(());
        }
    };

    // 遮罩点击处理
    let handle_overlay_click = move |evt: Event| {
        if !close_on_overlay {
            return;
        }
        // 仅当点击的是 overlay 本身时关闭（而非 modal 内容）
        let target = evt
            .target()
            .and_then(|t| t.dyn_into::<web_sys::HtmlElement>().ok());
        let current_target = evt
            .current_target()
            .and_then(|t| t.dyn_into::<web_sys::HtmlElement>().ok());

        if let (Some(target), Some(current)) = (target, current_target) {
            if target == current {
                close_modal.call(());
            }
        }
    };

    // 确认按钮点击
    let handle_confirm = move |_| {
        if let Some(cb) = on_confirm {
            cb.call(());
        }
        close_modal.call(());
    };

    // 关闭按钮点击
    let handle_close = move |_| {
        close_modal.call(());
    };

    let variant_class = variant.css_class().to_string();

    view! {
        <Show
            when=move || mounted.get()
            fallback=|| ()
        >
            <div
                class=move || {
                    if active.get() {
                        format!("modal-overlay active")
                    } else {
                        "modal-overlay".to_string()
                    }
                }
                on:click=handle_overlay_click
                on:keydown=handle_keydown
                role="dialog"
                aria-modal="true"
                aria-hidden=move || (!show.get()).to_string()
            >
                <div
                    class=format!("modal-container {}", variant_class)
                    on:click=|evt| evt.stop_propagation()
                >
                    // ===== Header =====
                    {move || {
                        title.as_ref().map(|t| {
                            view! {
                                <div class="modal-header">
                                    <h2 class="modal-title">{t.clone()}</h2>
                                    <button
                                        class="modal-close-btn"
                                        on:click=handle_close.clone()
                                        aria-label="关闭"
                                    >
                                        "×"
                                    </button>
                                </div>
                            }
                        })
                    }}

                    // ===== Body =====
                    <div class="modal-body">
                        {children()}
                    </div>

                    // ===== Footer =====
                    {move || {
                        if show_footer {
                            view! {
                                <div class="modal-footer">
                                    <button
                                        class="btn btn-ghost"
                                        on:click=handle_close.clone()
                                    >
                                        "取消"
                                    </button>
                                    {move || {
                                        if show_confirm {
                                            view! {
                                                <button
                                                    class={confirm_class.clone()}
                                                    on:click=handle_confirm
                                                >
                                                    {confirm_text.clone()}
                                                </button>
                                            }.into_view()
                                        } else {
                                            ().into_view()
                                        }
                                    }}
                                </div>
                            }.into_view()
                        } else {
                            ().into_view()
                        }
                    }}
                </div>
            </div>
        </Show>
    }
}

// ===== 辅助函数 =====

/// 锁定/解锁 body 滚动
fn lock_body_scroll(lock: bool) {
    let window = web_sys::window().unwrap();
    let document = window.document().unwrap();
    let body = document.body().unwrap();
    let style = body.style();

    if lock {
        style.set_property("overflow", "hidden").ok();
    } else {
        style.remove_property("overflow").ok();
    }
}

/// 在下一帧执行回调（用于 CSS transition 生效）
fn request_animation_frame(callback: impl FnOnce() + 'static) {
    let closure = Closure::once_into_js(callback);
    web_sys::window()
        .unwrap()
        .request_animation_frame(closure.as_ref().unchecked_ref())
        .unwrap();
}

/// 延时执行
fn set_timeout(callback: impl FnOnce() + 'static, duration: std::time::Duration) {
    let closure = Closure::once_into_js(callback);
    web_sys::window()
        .unwrap()
        .set_timeout_with_callback_and_timeout_arguments_and_0(
            closure.as_ref().unchecked_ref(),
            duration.as_millis() as i32,
        )
        .unwrap();
}
```

```rust
/// 内嵌的自定义表单组件 —— 演示 Modal 中嵌入复杂交互
#[component]
fn CustomForm(on_submit: Callback<String>) -> impl IntoView {
    let (name, set_name) = create_signal(String::new());
    let (email, set_email) = create_signal(String::new());

    let handle_submit = move |evt: web_sys::SubmitEvent| {
        evt.prevent_default();
        let n = name.get();
        if !n.is_empty() {
            on_submit.call(n);
        }
    };

    view! {
        <form on:submit=handle_submit style="display: flex; flex-direction: column; gap: 16px;">
            <div>
                <label style="display: block; color: #999; font-size: 12px; margin-bottom: 6px; text-transform: uppercase; letter-spacing: 0.08em;">
                    "姓名"
                </label>
                <input
                    type="text"
                    placeholder="输入你的姓名"
                    prop:value=name
                    on:input=move |evt| set_name.set(event_target_value(&evt))
                    style="
                        width: 100%;
                        padding: 10px 14px;
                        background: rgba(255,255,255,0.04);
                        border: 1px solid rgba(255,255,255,0.1);
                        border-radius: 8px;
                        color: #e0e0e0;
                        font-size: 14px;
                        outline: none;
                        transition: border-color 0.15s ease;
                        box-sizing: border-box;
                    "
                    on:focus=move |evt: web_sys::FocusEvent| {
                        let el = evt.target().unwrap().dyn_into::<web_sys::HtmlInputElement>().unwrap();
                        el.style().set_property("border-color", "rgba(99, 102, 241, 0.5)").ok();
                    }
                    on:blur=move |evt: web_sys::FocusEvent| {
                        let el = evt.target().unwrap().dyn_into::<web_sys::HtmlInputElement>().unwrap();
                        el.style().set_property("border-color", "rgba(255,255,255,0.1)").ok();
                    }
                />
            </div>

            <div>
                <label style="display: block; color: #999; font-size: 12px; margin-bottom: 6px; text-transform: uppercase; letter-spacing: 0.08em;">
                    "邮箱"
                </label>
                <input
                    type="email"
                    placeholder="your@email.com"
                    prop:value=email
                    on:input=move |evt| set_email.set(event_target_value(&evt))
                    style="
                        width: 100%;
```

## 6. 架构要点解析

### 6.1 动画挂载时序

这是 CSR 模态框最关键的实现细节：

```
show = true
    │
    ▼
set_mounted(true)      ← DOM 挂载，overlay 存在但 opacity: 0
    │
    ▼
requestAnimationFrame  ← 等浏览器完成一帧渲染
    │
    ▼
set_active(true)       ← 添加 .active class，CSS transition 播放入场动画
```

```
show = false
    │
    ▼
set_active(false)      ← 移除 .active class，CSS transition 播放退场动画
    │
    ▼
setTimeout(300ms)      ← 等动画结束
    │
    ▼
set_mounted(false)     ← 从 DOM 中移除
```

如果直接 `Unmount` 而不等待动画，退场动画将不会播放。

### 6.2 遮罩点击的事件过滤

```rust
// 点击事件绑定在 overlay（外层 div）上
// 通过 event.target === event.currentTarget 判断
// 确保只在点击遮罩本身时触发，而非点击 modal 内容区域
on:click=handle_overlay_click  // 绑定在 overlay div
on:click=|evt| evt.stop_propagation()  // modal-container 上阻止冒泡
```

### 6.3 Body 滚动锁定

当 Modal 打开时设置 `body { overflow: hidden }`，关闭时恢复，防止背景页面在 Modal 打开时仍可滚动。

### 6.4 无障碍支持

- `role="dialog"` 标识对话框角色
- `aria-modal="true"` 告知屏幕阅读器这是模态对话框
- `aria-hidden` 与可见状态联动
- `aria-label="关闭"` 为关闭按钮提供可读标签
- Esc 键盘事件关闭

## 7. 进阶扩展

### 7.1 全局 Modal 管理器（Context 注入）

当应用中有多个模块需要弹窗时，可以通过 Leptos 的 Context 机制实现全局管理：

```rust
/// 全局 Modal 上下文
#[derive(Clone)]
pub struct ModalManager {
    pub show: ReadSignal<bool>,
    pub set_show: WriteSignal<bool>,
    pub content: ReadSignal<ModalContent>,
    pub set_content: WriteSignal<ModalContent>,
}

#[derive(Clone)]
pub enum ModalContent {
    None,
    Confirm {
        title: String,
        message: String,
        on_confirm: Option<Callback<()>>,
    },
    Custom(ViewFn),
}

/// 在 App 顶层提供 ModalManager
#[component]
pub fn ModalProvider(children: Children) -> impl IntoView {
    let (show, set_show) = create_signal(false);
    let (content, set_content) = create_signal(ModalContent::None);

    let manager = ModalManager {
        show,
        set_show,
        content,
        set_content,
    };

    provide_context(manager.clone());

    // 渲染全局 Modal
    let manager_clone = manager.clone();

    view! {
        {children()}

        // 全局 Modal 渲染逻辑由 manager 驱动
        // ...
    }
}
```

子组件中使用：

```rust
#[component]
fn SomeChild() -> impl IntoView {
    let modal = use_context::<ModalManager>().expect("ModalProvider not found");

    let handle_delete = move |_| {
        modal.set_content.set(ModalContent::Confirm {
            title: "确认删除".into(),
            message: "此操作不可撤销".into(),
            on_confirm: Some(Callback::new(move |_| {
                // 执行删除逻辑
            })),
        });
        modal.set_show.set(true);
    };

    view! {
        <button on:click=handle_delete>"删除"</button>
    }
}
```

### 7.3 与 SSR 模式的差异

| 特性 | CSR | SSR |
|---|---|---|
| 初始渲染 | WASM 在浏览器执行 | HTML 由服务端生成 |
| `web-sys` API | 直接可用 | 仅在 `on_mount` / hydration 后可用 |
| Modal 挂载 | 通过 `mount_to_body` | 需要 `<Portal>` 或 hydration 策略 |
| 首屏性能 | 需下载 WASM | 首屏 HTML 即时可见 |
